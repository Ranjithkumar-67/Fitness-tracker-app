## ğŸ”§ Troubleshooting

### Common Issues

**Q: Data disappeared after closing browser**

A: Your data is saved in localStorage. Check:
1. Using same browser?
2. Private/Incognito mode? (data doesn't save)
3. Cleared browser data recently?

**Solution:**
- Use same browser always
- Backup data weekly (Excel export)
- Don't use incognito mode

**Q: Excel export not working**

A: Check:
1. Is XLSX library loaded? (check browser console)
2. Browser allows downloads?
3. Enough data to export?

**Solution:**
- Try different browser (Chrome recommended)
- Allow pop-ups and downloads
- Log at least 1 week of data first

**Q: Charts not showing**

A: Check:
1. Is Chart.js library loaded?
2. Enough data points? (need 2+ weights)
3. Browser console errors?

**Solution:**
- Refresh page
- Try different browser
- Log more data points

**Q: Email reports not sending**

A: Check:
1. EmailJS configured correctly?
2. Service ID, Template ID correct?
3. Email set in Settings?
4. Internet connection?

**Solution:**
- Verify all credentials
- Test in EmailJS dashboard
- Check spam folder
- See EMAIL_SETUP_GUIDE.md

**Q: Reminders not appearing**

A: Check:
1. Notification permission granted?
2. Browser supports notifications?
3. Computer notifications enabled?

**Solution:**
- Grant notification permission
- Enable in browser settings
- Enable in OS settings

**Q: Can't upload photos**

A: Check:
1. File size < 5MB?
2. Format is JPG/PNG?
3. Browser allows file access?

**Solution:**
- Compress large photos
- Use supported formats
- Allow file access in browser

### Data Recovery

**If you lost data:**

1. **Check other browsers** - might be in different browser
2. **Check backups** - Excel exports or JSON backups
3. **Check email** - weekly reports have some data
4. **Start fresh** - if necessary, begin new journey

**Prevent data loss:**
- Export to Excel weekly
- Backup JSON monthly
- Use same browser
- Don't clear browser data
